Posted on 2020-03-10

DOCKER + MySQL

1. Pull image

docker pull mysql

2. Check images

sudo docker images

3. Run image in a container

sudo docker run -p 3306:3306 --name mysql -e MYSQL_ROOT_PASSWORD=123456 -d mysql

4. Check status

docker container ls

5. Execute the bash of this container

sudo docker exec -it mysql bash

6. Run MySQL

mysql -uroot -p123456

7. SQL

CREATE DATABASE XXX;
drop database XXX;
use database XXX;

8. Save Docker Image

docker commit afcaf46e8305 centos-vim
docker IMAGE_ID DOCKER_ID/imagename:version
docker push DOCKER_ID/imagename
